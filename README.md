Xəbərdarlıq

Bu layihə yalnız təhsil, öyrənmə və kibertəhlükəsizlik sahəsində texniki biliklərin artırılması məqsədi ilə hazırlanmışdır.

Layihənin hər hansı formada qanunsuz, icazəsiz və ya zərərli məqsədlərlə istifadəsi yolverilməzdir.

Müəllif bu proqram təminatının üçüncü şəxslər tərəfindən səhv, etikaya zidd və ya qanunlara zidd istifadəsinə görə məsuliyyət daşımır.

Layihədən istifadə edən hər bir şəxs yerli və beynəlxalq qanunvericiliyə riayət etməyə şəxsən məsuliyyət daşıyır.
